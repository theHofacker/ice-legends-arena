## **üöÄ QUICK START**

### **Step 1: Create the 2.5D Scene**
1. In Unity menu: **Tools ‚Üí Create 2.5D Scene Setup**
2. This creates: `Assets/Scenes/Gameplay2.5D.unity`
3. Scene includes:
   - Isometric camera (orthographic, size 15)
   - Organized hierarchy (Environment, Gameplay, UI, Managers)
   - Placeholder notes for next steps

### **Step 2: Port Puck from Sandbox**
1. Open `PhysicsSandbox` scene (your 2D scene)
2. Find `Puck` prefab in Assets/Prefabs
3. Drag into `Gameplay2.5D` scene under `Puck_Placeholder`
4. Delete placeholder, keep the real Puck

### **Step 3: Add Camera Script**
1. Select `Main Camera` in Gameplay2.5D scene
2. Add Component ‚Üí `Puck Follow Camera`
3. Assign the Puck transform to `puckTransform` field
4. Adjust settings:
   - Follow Speed: 10
   - Base Zoom: 15
   - Predictive Follow: ON

### **Step 4: Test Basic Movement**
1. Create temporary player sprite (any square/circle)
2. Add Rigidbody2D + CircleCollider2D
3. Copy PlayerController script from PhysicsSandbox
4. Test: Player moves, camera follows puck

## **üé® ART PIPELINE - BUDGET APPROACH**

### **Option 1: Midjourney + Blender (Recommended)**

**Workflow:**
1. **Midjourney**: Generate character concept art
   - Prompt: "robotic hockey player, isometric view, 45 degree angle, vibrant colors, game character, transparent background"
2. **Import to Blender**: Use as texture/reference
3. **Render 8 directions**: North, NE, East, SE, South, SW, West, NW
4. **Export sprite sheets**: 256x256 per frame

**Cost:** ~$10/month Midjourney

### **Option 2: Free AI Tools**

**Tools:**
- **Scenario.gg**: Game sprite generator (free tier)
- **Leonardo.ai**: Similar to Midjourney (free credits)
- **Stable Diffusion**: Local, completely free

### **Option 3: 3D to 2D Pipeline**

**Workflow:**
1. **Mixamo** (free): Download rigged character
2. **Blender** (free): Import, apply simple textures
3. **Render from isometric angle**: Export sprite sheets
4. **Benefit**: Consistent angles automatically

## **üêõ TROUBLESHOOTING**

**Camera doesn't follow puck:**
- Check puck has "Puck" tag
- Verify PuckFollowCamera script is attached to Main Camera
- Assign puckTransform in Inspector

**Physics feels different:**
- Copy exact Rigidbody2D settings from PhysicsSandbox
- Ensure Physics2D settings are identical
- Check gravity scale = 0

**Sprites look wrong:**
- Verify Pixels Per Unit = 100
- Check sprite pivot is set to Center
- Ensure Sorting Layer is correct
