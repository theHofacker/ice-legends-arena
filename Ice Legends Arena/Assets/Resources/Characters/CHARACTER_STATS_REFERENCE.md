# Character Stats Reference

This document defines the 8 characters for Ice Legends Arena.
Use this as a guide when creating CharacterData ScriptableObject assets in Unity.

## How to Create Character Assets in Unity:
1. In Project window, navigate to `Assets/Resources/Characters/`
2. Right-click ‚Üí Create ‚Üí Ice Legends ‚Üí Character Data
3. Name it after the character (e.g., "Elementalist_Blaze")
4. Fill in stats according to this reference
